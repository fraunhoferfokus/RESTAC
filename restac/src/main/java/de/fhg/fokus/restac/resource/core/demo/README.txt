Package de.fhg.fokus.restac.resource.core.demo contains sample classes 
for the demonstration of the resource model of RESTAC.

How to use:

Start MyResourceMessageHandler first, then either the ClientTestClass or
the ResourceProxyClientTestClass.

To test chunked encoding start first ChunkEnabledMyResourceMessageHandler then
ChunkEnabledClientTestClass.
